# kolkata-bus-gtfs
Get GTFS Format for bus transport routes in Kolkata City. 

# Why This Exists
Kolkata has one of the largest urban bus networks in India, yet no clean, publicly available structured dataset of its routes exists. Without this data, building transit apps, conducting transport research, or integrating Kolkata buses into tools like Google Maps is impossible.
This project creates that foundation â€” starting with manually verified route and stop data, structured in GTFS format, the international standard used by every major transit app in the world.

# How to Contribute Data
No coding experience needed. Just ride a bus.

Open the collector tool at neelabho.github.io/kolkata-bus-gtfs
Fill in the route details before boarding
Tap Add Stop at every bus stop along the route
Tap End Trip when you reach the final stop
Download the trip JSON file
Send it to Neelabho on WhatsApp (if you have the number) or email: examplesir6@gmail.com
